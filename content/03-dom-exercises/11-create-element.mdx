# Task

Use Javascript to create an element that looks like this:

```html
<h2 id="heading-2">Hand-crafted header</h2>
```

And insert it inside the `div.exercise`.

# HTML

```html
<div class="exercise">
  <!-- your element here -->
</div>
```

# Code Sandbox
<Sandbox src="/dom-11" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const h2 = document.createElement("h2")
h2.id = "heading-2"
h2.textContent = "Hand-crafted header"
const parent = document.querySelector(".exercise")
parent.append(h2)
```

</details>
