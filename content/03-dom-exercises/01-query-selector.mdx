# Task

Find the `<h3>` tag and change its text to say `Hello!`

# HTML

```html
<div class="exercise">
  <h3>Change My Text</h3>
</div>
```

# Code Sandbox
<Sandbox
  html={`
<div class="exercise">
  <h3>Change My Text</h3>
</div>
  `}
  test={`
describe('h3', function () {
  it('should have the text "Hello!"', function () {
    const h3 = document.querySelector("h3")
    chai.expect(h3.textContent).to.equal("Hello!");
  });
});
  `}
/>

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const h3 = document.querySelector("h3")
h3.textContent = "Hello!"
```

</details>
