# Task

Find the `<h3>` tag and change its text to say `Hello!`

# HTML

```html
<div class="exercise">
  <h3>Change My Text</h3>
</div>
```

# Code Sandbox
<Sandbox src="/dom-01.html" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const h3 = document.querySelector("h3")
h3.textContent = "Hello!"
```

</details>
