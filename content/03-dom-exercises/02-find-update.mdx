# Task

Find the `<img>` tag and update it as follows:

- Display the image from this url: `https://randomfox.ca/images/25.jpg`
- change the `alt` text to read `A random cute fox`

# HTML

```html
<div class="exercise">
  <div class="image-holder">
    <h3>Look at this cute fox</h3>
    <img src="https://via.placeholder.com/300" alt="No fox yet :(" />
  </div>
</div>
```

# Code Sandbox
<Sandbox src="/dom-02.html" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const img = document.querySelector("img")
img.src = "https://randomfox.ca/images/25.jpg"
img.alt = "A random cute fox"
```

</details>
