# Task

Find the `<p>` tag with the id of `find-me` and update it as follows:

- Add a class of `primary`
- Change its font color to `blue`

# HTML

```html
<div class="exercise">
  <p>You can ignore this one</p>
  <p id="find-me">Find me</p>
  <p>You can ignore me too</p>
</div>
```

# Code Sandbox
<Sandbox src="/dom-03" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const pTag = document.querySelector("#find-me")
pTag.style.color = "blue"
pTag.className = "primary"
```

</details>
