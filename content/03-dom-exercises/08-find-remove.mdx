# Task

Find the `<h3>` tag and remove it from the page.

# HTML

```html
<div class="exercise">
  <h3>Now you see me...</h3>
</div>
```

# Code Sandbox
<Sandbox src="/dom-08" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const h3 = document.querySelector("h3")
h3.remove()
```

</details>
