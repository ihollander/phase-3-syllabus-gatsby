# Task

Find the `<ul>` tag with the class of `clear-me` remove all its child elements. Don't remove the `<ul>`, only the `<li>` elements inside it!

# HTML

```html
<div class="exercise">
  <ul class="clear-me">
    <li>pls delete</li>
    <li>remove me too</li>
    <li>no longer want</li>
  </li>
</div>
```

# Code Sandbox
<Sandbox src="/dom-10" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const ul = document.querySelector(".clear-me")
ul.innerHTML = ""
```

</details>


<details>
  <summary>Click to show another solution!</summary>

```javascript
const ul = document.querySelector(".clear-me")
while(ul.firstChild) {
  ul.firstChild.remove()
}
```

</details>
