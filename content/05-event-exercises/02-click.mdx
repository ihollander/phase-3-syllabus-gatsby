# Task

Find the `<span>` tag inside the `<button>`. Use Javascript to increment that number by 1, and display it in the span.

Make sure to read the current number of likes from span and convert it to an integer before updating the DOM!

# HTML

```html
<div class="exercise">
  <button>
    Likes: <span class="likes-count">11</span>
  </button>
</div>
```

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const button = document.querySelector("button")
button.addEventListener("click", increment)

function increment() {
  const span = document.querySelector(".likes-count")
  const likes = parseInt(span.textContent) + 1
  span.textContent = likes
}
```
