# Task

Find the `<button>` tag and add an event listener for a 'click' event. When the button is clicked, toggle the class of `dark-mode` on and off on the `div.exercise` element.

# HTML

```html
<div class="exercise">
  <h1>Hello!</h1>
  <p>Here's some text</p>
  <button>Toggle Dark Mode</button>
</div>
```

# Code Sandbox
<Sandbox src="/events-01.html" />

# Solution

<details>
  <summary>Click to show solution!</summary>

```javascript
const exercise = document.querySelector(".exercise")
const button = exercise.querySelector("button")
button.addEventListener("click", function() {
  exercise.classList.toggle("dark-mode")
})
```

</details>
